# X-Tiger Core

X-Tiger Core is the functionality plugin for the X-Tiger WordPress platform.

It is separate from the X-Tiger block theme.

## Responsibilities

- settings;
- contacts;
- social links;
- schema;
- dynamic Gutenberg blocks;
- integrations;
- AI-readable manifest.

## Current state

This repository currently contains only the minimal plugin skeleton.

No custom blocks, settings UI, schema output, or build step are included yet.

## Theme relationship

The X-Tiger theme owns the visual foundation. Future plugin blocks should reuse theme foundation classes when they are intended for the X-Tiger theme.
